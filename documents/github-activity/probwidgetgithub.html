GitHub Activity Stream

GitHub Activity Stream an embeddable widget that displays a user's public activity on GitHub. It displays a user's username, name, photo, and a specified number of recent activity updates.

Visit the project repository on GitHub for instructions on how to embed it on your site. This project is forked from [https://github.com/caseyscarborough/github-activity](https://github.com/caseyscarborough/github-activity)
